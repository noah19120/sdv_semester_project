# Integrating SDV Architecture on an AVL Robotic Truck
This Github is used to share our work on the semester project : Integrating SDV Architecture on an AVL Robotic Truck

## truck_architecture
Contain a guide about the architecture of the AVL Truck we worked with during our project and a test guide. The test results can be find in the folder docs

## CarSim_example
CarSim example contain the code and the guide for the examples we runned during this project : The classic CarSim example from Eclipse SDV and the modified version using a ROS2 bridge

## semester_project_sdv_report.pdf
This file contain the pdf version of our report on our work during this project
